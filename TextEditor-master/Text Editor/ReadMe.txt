PLEASE NOTE!
=============
* If your system suspects for virus detections, please ignore as this is just a false alarm. The program and all the file contents are 100% safe.

=============
TO INSTALL
=============
1. Open "setup.exe".
2. Follow all the instructions to complete the Text Editor installation.
3. This causes the program to be installed into your system.
4. To search for the program in your system, simply search for "Text Editor" and run it.

=============
TO UNINSTALL
=============
1. Direct to your program removal/ program uninstallation utility tool. For example, in Windows it is called "Add or remove programs".
2. Inside the removal utility tool, search for "Text Editor", and complete the uninstallation steps.
3. The program will then be completely removed from your system.
